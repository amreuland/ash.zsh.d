# environment configuration

configuration files for various parts of the environment

- [burrow/](burrow/) - extra plugin configurations
- [den/](den/) - environment base configurations
- [git/](git/) - git configs
- [i3/](i3/) - i3 window manager config
- [neofetch/](neofetch/) - neofetch fetch tool config
- [tmux/](tmux/) - tmux config
- [vim/](vim/) - vim config scripts
- [zsh/](zsh/) - zsh entry points
